## 📖 Project Overview

This project is a fully responsive, multi-section website for a barbershop brand called **Fade Masters**. It was designed and developed to simulate a real-world client project, focusing on:

- Clean UI/UX design
- Component-based architecture
- Responsive layouts across all devices
- Professional code organization

The goal of this project is to demonstrate **frontend development skills** using React and CSS, while building something practical and portfolio-ready.

---

## ✨ Features

### 🎯 Core Features

- Smooth scrolling navigation between sections
- Fully responsive design (mobile, tablet, desktop)
- Modular and reusable React components
- Clean and structured CSS using BEM methodology

### 💡 UI/UX Features

- Hero section with background image and call-to-action
- Services section with feature cards
- Image gallery showcasing work
- About section highlighting brand identity
- Contact form with validation and user feedback
- Footer with navigation and social links

### 🎨 Design Highlights

- Modern dark theme with gold accent color
- Consistent typography using **Bebas Neue** and **Poppins**
- Hover effects and transitions for interactivity
- Clean spacing and layout hierarchy
